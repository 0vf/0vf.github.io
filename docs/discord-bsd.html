Discord-BSD installation instructions

Discord-BSD is a native Discord client made to run on FreeBSD smoothly.

Installation

First, install the required dependencies and clone the repo:

sudo pkg install electron19 node16 npm-node16
git clone https://github.com/z-ffqq/Discord-BSD.git

Then, change your directory to the clond repository

cd Discord-BSD

Finally, launch the install script:

./install.sh

After running these commands, the client should appear in your app launcher as usual.

Mobile isn't supported in documentation
